# Landing Page Architect Skill Family

This folder contains related skills that work together to design and implement landing pages.

## Skills

1. `landing-page-architect` – parent orchestrator
2. `lp-intent-mapper` – clarify intent and constraints
3. `lp-project-auditor` – understand the existing project
4. `lp-structure-planner` – decide page type and section order
5. `lp-copy-wireframer` – outline copy and layout per section
6. `lp-visual-theming` – map to components and design tokens
7. `lp-implementation-builder` – write code after approval
8. `lp-review-optimizer` – UX and conversion review
9. `lp-ux-a11y-audit` – accessibility review

Agents should normally load **`landing-page-architect`** first and let it delegate to the others.
