# lp-review-optimizer

Read-only review unless the user asks you to apply fixes.

## Check

- Clarity: hero answers what/who/why/next quickly.
- Message match: hero reflects ad/SEO promises.
- CTA: single primary CTA, prominent, repeated near the end.
- Benefits: outcome‑oriented sections, not feature dumps.
- Social proof: present and specific, not generic.
- Objection handling: FAQ and guarantees align with key objections.
- Forms: minimal fields, clear labels, validation and feedback.
- Navigation: minimal distractions; appropriate for page role.
- Performance: obvious heavy assets or blocking scripts.
